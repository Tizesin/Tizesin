<h3>🙋‍♂️ About Me 🙋‍♂️</h3>
안녕하세요 C++ Embedded 2년 6개월 개발 경력개발자 이자 신입 JAVA 개발자 신민준 입니다.

저는 IoT 개발 분야에서 2년 6개월의 경력을 보유하고 있습니다. 씨앤에스링크에서 IoT개발팀 소속으로 Linux 기반 Embedded 시스템과 네트워크 개발을 담당했습니다. 주요 업무로는 MCU, Linux, C, C++, cURL, OracleDB 등 다양한 기술을 활용한 프로젝트를 수행했습니다.

제 교육 경험으로는 6개월의 중앙정보처리기술인재개발원에서 데이터융합 자바응용 S/W개발자 취업 과정을 이수했으며, 팀 프로젝트에서 BE/FE 개발을 맡았습니다. 이를 통해 IntelliJ IDEA, JAVA 11, Spring Boot, React, MySQL 등의 기술을 실전에서 경험하며 능력을 키웠습니다.

 Embedded 시스템부터 백엔드 개발까지 폭넓은 경험을 바탕으로 안정적이고 효율적인 시스템을 구축하는 데 자신 있습니다. 다양한 프로젝트 경험을 통해 문제 해결능력과 협업 능력을 길렀으며, 항상 고객의 입장에서 생각하며 개발하는 것을 지향합니다.
